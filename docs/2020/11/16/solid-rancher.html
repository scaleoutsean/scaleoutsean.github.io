solid-rancher

16 Nov 2020 - Less than 1 minute read

solid-rancher is a Github repo I created to organize and enrich information related to Rancher on SolidFire (and NetApp HCI in general).

A non-mission is to substitute or duplicate information from the official documentation (whether from Rancher or from NetApp).

Like with other repos I maintain, the idea is to share useful information quickly and in an accessible way. Quality contributions are welcome (submit issue on Github).

This repo is experimentally published as a Web site located at https://scaleoutsean.github.io/solid-rancher.

UPDATE (September 2021)

Because most new SolidFire deployments in Kubernetes environments include all sorts of distributions and servers, I spun up a new, generic repository for Kubernetes with SolidFire, which focuses on NetApp Trident and SolidFire.

Please use https://solidfire-kubernetes.pages.dev instead of solid-rancher.
